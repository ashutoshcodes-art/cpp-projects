# 🏧 ATM Simulation System (C++)

A console-based ATM Simulation System developed in **C++**. This project allows users to perform basic banking operations through a simple menu-driven interface, including checking account balance, depositing money, and withdrawing money. It demonstrates the use of functions, conditional statements, and pass-by-reference in C++.

---

## Features

- Check account balance
- Deposit money
- Withdraw money
- Balance updates after each transaction
- Deposit amount validation
- Withdrawal amount validation
- Menu-driven interface
- Session termination option

---

## Concepts Used

- Functions
- Function Prototypes
- Pass by Reference
- Pass by Constant Reference
- Loops
- Conditional Statements
- Input Handling
- Menu-Driven Programming

---

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o atm_system
```

### Run

```bash
./atm_system
```
