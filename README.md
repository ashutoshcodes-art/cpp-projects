| Project | Description |
|---------|-------------|

# ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/ashutoshcodes-art/cpp-projects.git
```

Open any project folder.

Compile the source file:

```bash
g++ main.cpp -o app
```

Run:

```bash
./app
```
