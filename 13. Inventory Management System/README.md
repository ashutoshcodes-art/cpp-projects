# 📦 Inventory Management System (C++)

A console-based Inventory Management System developed in **C++**. This project allows users to manage product records through a menu-driven interface. It supports adding, updating, deleting, searching, restocking, and selling products while storing data permanently using file handling.

---

## Features

- Add new products
- View all products
- Search products by ID
- Update product details
- Delete products
- Restock product quantity
- Sell products with automatic bill calculation
- Unique Product ID validation
- Data persistence using file handling
- Formatted product display
- Menu-driven interface
- Input validation for numeric values

---

## Concepts Used

- Structures (`struct`)
- Vectors (`std::vector`)
- Functions
- File Handling (`ifstream`, `ofstream`)
- String Handling
- Loops
- Conditional Statements
- Input Validation (`cin.fail()`)
- Searching (Linear Search)
- Formatted Output (`iomanip`)

---

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o inventory_management
```

### Run

```bash
./inventory_management
```
