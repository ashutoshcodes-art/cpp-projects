# 🔤 String Operations System (C++)

A console-based String Operations System developed in **C++**. This project provides a menu-driven interface to perform various string manipulation operations such as concatenation, comparison, reversing, palindrome checking, character searching, and case conversion. It demonstrates the use of strings, functions, loops, and built-in string algorithms in C++.

---

## Features

- Enter a string
- Display the current string
- Find string length
- Copy a string
- Concatenate two strings
- Compare two strings
- Reverse a string
- Check whether the string is a palindrome
- Count vowels and consonants
- Convert string to uppercase
- Convert string to lowercase
- Search for a character
- Replace a character
- Menu-driven interface
- Input validation for menu selection

---

## Concepts Used

- Strings (`std::string`)
- Functions
- Function Parameters (Pass by Value & Reference)
- Loops
- Conditional Statements
- Character Functions (`toupper()`, `tolower()`, `isalpha()`)
- STL Algorithm (`reverse()`)
- Input Validation (`cin.fail()`)
- String Manipulation

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o string_operations
```

### Run

```bash
./string_operations
```
