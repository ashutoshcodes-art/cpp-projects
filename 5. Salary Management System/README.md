# 💰 Salary Management System (C++)

A console-based Salary Management System developed in **C++**. This project allows users to manage employee records through a menu-driven interface. It supports adding employees, viewing employee details, searching employees, updating salaries, and deleting employee records. The project demonstrates the use of structures, vectors, strings, and functions in C++.

---

## Features

- Add new employees
- View all employee records
- Search employee by Employee ID
- Update employee salary
- Delete employee records
- Unique Employee ID validation
- Store Employee Name, ID, Department, and Salary
- Dynamic storage using `std::vector`
- Menu-driven interface

---

## Concepts Used

- Structures (`struct`)
- Vectors (`std::vector`)
- Functions
- Strings (`std::string`)
- Loops
- Conditional Statements
- Linear Search
- Dynamic Data Storage
- Input Handling (`getline()`, `cin.ignore()`)

---

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o salary_management
```

### Run

```bash
./salary_management
```
