# 🛒 Shop Management System (C++)

A console-based Shop Management System developed in **C++**. This project manages product records using arrays and provides a menu-driven interface to perform essential shop operations such as adding products, updating stock, selling products, searching products, and deleting records. It demonstrates the use of arrays, structures, strings, and functions.

---

## Features

- Add new products
- View all products
- Search products by ID
- Update product stock
- Sell products with automatic bill calculation
- Delete products
- Unique Product ID validation
- Formatted product display
- Menu-driven interface
- Input validation for integer values

---

## Concepts Used

- Arrays
- Structures (`struct`)
- Strings (`std::string`)
- Functions
- Function Parameters (Pass by Value & Reference)
- Loops
- Conditional Statements
- Linear Search
- Input Validation (`cin.fail()`)
- Formatted Output (`iomanip`)

---

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o shop_management
```

### Run

```bash
./shop_management
```
