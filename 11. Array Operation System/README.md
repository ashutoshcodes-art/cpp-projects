# 🔢 Array Operations System (C++)

A console-based Array Operations System developed in **C++**. This project provides a menu-driven interface to perform common array operations such as insertion, deletion, searching, sorting, reversing, and finding statistical values. It demonstrates the use of arrays, functions, loops, and basic algorithms in C++.

---

## Features

- Create an array
- Display array elements
- Insert an element at any position
- Delete an element
- Search for an element
- Update an existing element
- Sort the array (Bubble Sort)
- Reverse the array
- Find the maximum element
- Find the minimum element
- Calculate the sum of elements
- Calculate the average
- Remove duplicate elements
- Input validation for integer values
- Menu-driven interface

---

## Concepts Used

- Arrays
- Functions
- Function Parameters (Pass by Value & Reference)
- Loops
- Conditional Statements
- Searching (Linear Search)
- Bubble Sort
- Array Manipulation
- Input Validation (`cin.fail()`)
- Basic Algorithms

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o array_operations
```

### Run

```bash
./array_operations
```
