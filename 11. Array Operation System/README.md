# 📊 Array Operations System

A menu-driven console application developed in **C++** to perform various array operations. The program allows users to create an array and perform common operations such as insertion, deletion, searching, sorting, reversing, finding statistical values, and removing duplicate elements.

This project focuses on strengthening array manipulation techniques, algorithm implementation, and modular programming using functions.

---

## ✨ Features

- 📝 Create a new array
- 📋 Display array elements
- ➕ Insert an element at any position
- ❌ Delete an element from any position
- 🔍 Search an element
- ✏️ Update an existing element
- 🔃 Sort the array
- 🔄 Reverse the array
- 📈 Find the maximum element
- 📉 Find the minimum element
- ➕ Calculate the sum of all elements
- 📊 Calculate the average
- 🚫 Remove duplicate elements
- ✅ Input validation
- 📋 Menu-driven interface

## 🚀 How It Works

### Create Array

Creates an integer array by accepting the desired size and user input for each element.

---

### Display Array

Displays all array elements in their current order.

---

### Insert Element

Allows inserting a new element at any valid position by shifting subsequent elements.

---

### Delete Element

Deletes an element from a specified position and shifts remaining elements to maintain continuity.

---

### Search Element

Performs a **Linear Search** to locate a specified element and returns its index if found.

---

### Update Element

Replaces the value stored at a selected position with a new value.

---

### Sort Array

Sorts the array in ascending order using the **Bubble Sort** algorithm.

---

### Reverse Array

Reverses the array by swapping elements from both ends until the center is reached.

---

### Find Maximum

Traverses the array to determine the largest element.

---

### Find Minimum

Traverses the array to determine the smallest element.

---

### Sum of Elements

Calculates and displays the sum of all array elements.

---

### Average

Calculates the arithmetic mean of all elements.

---

### Remove Duplicates

Removes duplicate values by comparing each element with the remaining elements and shifting the array after deletion.
