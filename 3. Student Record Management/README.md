# 🎓 Student Record Management System (C++)

A console-based Student Record Management System developed in **C++**. This project allows users to manage student records through a menu-driven interface by adding, viewing, searching, and deleting student information. It demonstrates the use of structures, vectors, functions, and string handling in C++.

---

## Features

- Add new student records
- View all students
- Search student by Roll Number
- Delete student records
- Store Name, Roll Number, and CGPA
- Dynamic storage using `std::vector`
- Menu-driven interface

---

## Concepts Used

- Structures (`struct`)
- Vectors (`std::vector`)
- Functions
- Strings (`std::string`)
- Loops
- Conditional Statements
- Linear Search
- Dynamic Data Storage
- Input Handling (`getline()`, `cin.ignore()`)

---

## How to Compile and Run

### Compile

```bash
g++ main.cpp -o student_management
```

### Run

```bash
./student_management
```
